Notes on this theme:

  • This is the three column version of obscene_v3
  • Theme comes with templates... the templates have only minor differences to the default templates: 1~ adds rounded corners to the head of news items. 2~ adds the little images beside links in the main menu (I didnt do this with CSS because of a bug in I.E. 3~ changes the head of download and link items to a background color the same as news item head. for the images in menu to show, they must be placed in the xoops images folder /xoops-root/images/... Theme doesnt require the use of these templates, they only provide a few little asthetic improvements (in my oppinion) ... if you use templates without placing the link images in correct folder then your site will produce a 404 for those images. These templates will also work fine with Obscene_v2 but may or may not with other themes. consideration should be taken before deciding to use templates.
  • I have tested this theme in the following browsers: IE 5.1, 5.5 and 6, FireFox 0.93, Opera 7.54 & Mozilla 0.73. As always you should test a theme yourself in various browsers before using. The theme is pretty much uniform across ie6, Mozilla, Firefox and Opera. I haven't access to a MAC or linux system so I can only assume that theme works ok under these platforms.
  • I assume that most people will be able to modify the links in the top nav bar and the Logo etc to their liking. If you dont know any html or arent sure how to do this then drop me a line and provided I have time I will help ya out.
  • Theme has a "Theme by Astonthemes" link near footer of page, I would appreciatte this being left there but I dont demand it.

Important:

When you upload the theme, make sure you upload the right folder!!. In the zipping process another folder is usualy created, this means that after unzipping, the actual Theme folder may be inside another folder. The correct folder to upload is the folder that contains the "theme.html" file. If you get a blank site after installing the theme, then this is likely the problem.

To upload templates: goto administration/system admin/templates/ and you will see the option to upload a tar.gz file.

Templates will not take effect unless they are selected as default in aministration - general settings - Default Template Set.

You may have customized your own templates and wish to keep these customizations... below is source for modifications in these templates, you could copy and paste these where needed.
